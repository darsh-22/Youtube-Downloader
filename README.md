# Youtube-Downloader

YouTube Downloader is a Python library for dealing with word pluralization.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install pytube
pip install tkinter
```
## Run code

```
python main.py
```
